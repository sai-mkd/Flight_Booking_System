Flight Booking System

A full-stack web application developed using Java, Spring Boot,  Maven, and MySQL that enables users to search flights, book tickets, manage reservations, and view booking details through an intuitive web interface.

Features
1. User registration and authentication
2. Flight search and booking
3. Passenger information management
4. Booking history and cancellation
5. Flight schedule management

Future Enhancements
1. Payment gateway integration
2. Email notifications
3. Docker deployment
